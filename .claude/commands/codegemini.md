# CodeGemini 功能

啟動 CodeGemini 程式碼輔助功能。

功能選項：
1. **程式碼分析**：分析程式碼結構與品質
2. **測試生成**：自動生成單元測試
3. **文件生成**：生成程式碼文件
4. **程式碼審查**：AI 程式碼審查建議
5. **重構建議**：程式碼優化建議
6. **Codebase Embedding**：程式碼庫向量化搜尋

執行時：
- 顯示 CodeGemini 功能選單
- 載入相關模組（延遲載入）
- 提供互動式操作介面

以台灣地區繁體中文語境處理。
