# 影音功能選單

ChatGemini_SakiTool 的多媒體處理功能入口。

功能選項：
1. **圖片分析**：分析圖片內容、物體識別、場景描述
2. **圖片生成**：使用 Imagen 生成圖片
3. **影片分析**：分析影片內容、生成摘要
4. **影片生成**：使用 Veo 生成影片片段
5. **音訊處理**：音訊提取、合併、處理
6. **字幕生成**：自動生成影片字幕
7. **影片特效**：裁切、濾鏡、速度調整

執行時：
- 顯示互動式選單
- 根據使用者選擇載入對應模組
- 提供功能說明與使用指引

以台灣地區繁體中文語境處理。
