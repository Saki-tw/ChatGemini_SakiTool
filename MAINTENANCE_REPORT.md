## ⚠️ 已知問題

### 1. Windows 支援
**狀態**: 未完整測試
**影響**: INSTALL.sh 僅支援 macOS/Linux
**計畫**: 未來版本加入 Windows 支援

### 2. 大型影片處理
**狀態**: 長時間處理可能超時
**影響**: 超過 1 小時的影片可能需要分段處理
**暫時解決**: 使用較小的影片檔案（<30分鐘）

### 3. prompt-toolkit 相容性
**狀態**: 部分終端機不支援
**影響**: 進階文字介面功能無法使用
**解決**: 自動降級到標準輸入

### 4. 快取建立延遲
**狀態**: 首次建立快取需要 1-2 秒
**影響**: 使用者體驗略微延遲
**計畫**: 優化快取建立流程

## 🐛 已修復問題（v1.0.2）

### 1. 中文標點符號問題
**問題**: 中文標點符號顯示錯誤
**修復**: 統一使用 UTF-8 編碼
**版本**: v1.0.1

### 2. 檔案路徑空格問題
**問題**: 檔案路徑包含空格時讀取失敗
**修復**: 加入路徑引號處理
**版本**: v1.0.1

### 3. 快取門檻驗證
**問題**: 門檻值低於模型要求時崩潰
**修復**: 加入門檻值驗證與錯誤提示
**版本**: v1.0.3

### 4. 虛擬環境偵測
**問題**: 多版本虛擬環境偵測錯誤
**修復**: 智慧偵測 venv_py313, venv_py314 等
**版本**: v1.0.3
