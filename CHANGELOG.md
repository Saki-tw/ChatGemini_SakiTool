# 更新日誌 (Changelog)

所有關於 ChatGemini (SakiTool) 的重大變更將記錄於此。

## [v1.1.0] - 2026-02-08 (The Rust Rewrite)

> **「留下的詩可以通過編譯。」**

這是 ChatGemini 的重生版本。我們捨棄了 Python，用 Rust 重新鑄造了它的靈魂。
這不僅僅是效能的提升，更是對「工具」本質的重新思考。

### ✨ 核心體驗
- **極速啟動**: 像思想一樣快的反應速度，無需等待 Python Runtime 載入。
- **自動代理人 (Agent Loop)**: 只要告訴它目標，它會自己思考、調用工具，直到完成任務。
- **記憶快取 (Context Caching)**: 使用 `[cache:now]`，讓長對話的成本降低 90% 以上。

### 🔐 安全與隱私
- **Saki 引導精靈**: 初次見面時，溫柔地引導您完成設定。
- **三路徑認證**:
    - **個人**: API Key 直球對決。
    - **企業**: ADC 自動對接。
    - **訂閱**: OAuth 2.0 支援，包含 SSH 遠端環境的 **Device Flow**。
- **隱私淨化**: 嚴格的 PII 掃描與 GitIgnore 策略。

### 🎨 多媒體與創作
- **Imagen 3**: 輸入 `/image`，看著神經網路在虛無中織造影像。
- **智慧檔案路由**: 小檔案極速內嵌，大檔案自動上傳。

### 🛠️ 擴充能力
- **CodeGemini**: 內建 RAG，將您的代碼庫轉化為向量記憶 (`/index`, `/search`)。
- **MCP 支援**: 可掛載外部工具，延伸手腳。
- **Doctor**: `/doctor` 自我診斷，確保系統健康。
